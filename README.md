# My wayland compositor

A wayland compositor (tiling in future) written in C using wlroots library. I Building this project for learning purposes. Currently focused on adding tiling windows method and compositor logic improvements.

# Technologies

- C
- wlroots
- libwayland
- libinput
- Linux core

# Features

- scroll, resize, drag windows
- xwayland windows support(in future)
- tiling(in future)
- keybindings cfg file (in future)
